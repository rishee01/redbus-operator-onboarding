# 📌 Overview

The **redBus Operator Onboarding Portal** is a responsive web application designed to simplify the onboarding process for:

- 🚌 Bus Operators
- 🏢 Travel Agencies
- 🌍 Fleet Management Businesses

This platform provides a clean and scalable onboarding experience with modular frontend architecture and server-side rendering.

---

# ✨ Features

## 🎯 Core Features

- Responsive Landing Page
- Operator Registration Flow
- Reusable EJS Components
- Dynamic Server-side Rendering
- FAQ Interaction Logic
- Organized MVC-like Structure
- Asset Bundling using Webpack
- Fast Development Workflow

# 📂 Project Structure

```bash
redbus-operator-onboarding/
│
├── public/                 # Static assets
│
├── server/
│   ├── controllers/        # Request handlers
│   ├── models/             # Data models
│   └── routes/             # Express routes
│
├── src/
│   ├── css/                # Stylesheets
│   └── js/                 # Frontend logic
│
├── views/
│   ├── partials/           # Reusable EJS components
│   └── index.ejs           # Main landing page
│
├── server.js               # Main Express server
├── webpack.config.js       # Webpack configuration
├── package.json            # Dependencies & scripts
└── README.md
```

# ⚙️ Installation & Setup

## 📥 Clone Repository

```bash
git clone https://github.com/rishee01/redbus-operator-onboarding.git
```

## 📁 Navigate to Project

```bash
cd redbus-operator-onboarding
```

## 📦 Install Dependencies

```bash
npm install
```

---

# 🚀 Run Project

## Start Production Server

```bash
npm start
```

Server runs on:

```bash
http://localhost:3000
```

---

# 🛠️ Development Mode

## Run Webpack Watcher

```bash
npm run watch
```

## Run Nodemon Server

```bash
npm run dev
```
